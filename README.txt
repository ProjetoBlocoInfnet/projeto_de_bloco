Passos para primeira execução do projeto

1 - Criar o banco questionario_db no seu mysql
2 - Configurar o wildfly usando o arquivo standalone.xml que consta na pasta Util.
3 - Colocar mysql-connector-java-5.1.37-bin.jar e module.xml dentro da pasta \modules\system\layers\base\com\mysql\main\ dentro do wildfly.
4 - Importar e executar os projetos QuestionarioInfnet*
5 - Após execução, executar os SQLs do arquivo sqls_start_base.sql que consta na pasta Util.
6 - Para logar no sistema, use o login adm e a senha adm.
7 - Para o teste do webservice, vá no menu Áreas -> Usuários e cadastre alguns alunos.

Have fun!!!
